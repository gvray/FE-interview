# 计算机网络

网络是前端面试的必考基础。从浏览器输入 URL 到页面渲染，HTTP 缓存、TCP 握手、HTTPS、跨域、WebSocket 等几乎每轮面试都会涉及。能否讲清"一次请求在网络上到底发生了什么"，直接体现候选人的工程深度。

## 内容

- [网络协议 FAQ](./FAQ.md) - 报文结构、状态码、缓存策略（强缓存/协商缓存）、HTTPS、HTTP/2、跨域（CORS）等高频问题。
- [HTTP 缓存详解](./http-cache.md) - 强制缓存与协商缓存的完整流程。

## 学习路径建议

1. **HTTP 优先**：把请求/响应报文、常见状态码、缓存（`Cache-Control`、`ETag`）彻底吃透。
2. **安全与加密**：理解 HTTPS 的握手流程、对称与非对称加密、证书链，能讲清"为什么 HTTPS 安全"。
3. **跨域与通信**：CORS 预检、同源策略、WebSocket、SSE，以及跨域解决方案的取舍。
4. **结合浏览器**：把网络与浏览器渲染、Service Worker、CDN 结合起来，形成"从输入 URL 到看到页面"的完整链路。
