# AI 开发专栏

本专栏系统介绍人工智能技术在前端开发中的应用，从基础概念到实战应用，帮助开发者掌握 AI 开发技能。

## 📚 学习路径

### 🌱 第一阶段：AI 基础知识
> 建立扎实的理论基础

- [AI 基础概念](./basics/01-introduction.md) - 人工智能概述与发展历程
- [机器学习基础](./basics/02-machine-learning.md) - 核心算法与实际应用
- [深度学习入门](./basics/03-deep-learning.md) - 神经网络与架构设计
- [大语言模型基础](./basics/04-large-language-models.md) - Transformer 与预训练技术

### 🚀 第二阶段：应用开发实践
> 掌握实际开发技能

- [AI 应用开发入门](./app-development/01-introduction.md) - Prompt、Agent 与 Function Calling
- [LangChain 框架](./app-development/langchain/01-introduction.md) - LLM 应用开发框架

### 💎 第三阶段：原理深入
> 理解底层技术原理

- [大模型原理](./llm-fundamentals/) - 深入理解 LLM 底层技术

## 🎯 专栏特色

- **系统化**：从基础到进阶的完整学习路径
- **实战性**：结合实际项目案例
- **前沿性**：紧跟 AI 技术发展趋势
- **专业性**：严谨的技术内容与最佳实践

## 📖 适用人群

- 前端开发者想转型 AI 开发
- 需要集成 AI 功能的应用开发者
- 对大模型应用感兴趣的技术人员
- 希望系统学习 AI 开发的工程师

## 🛠️ 技术栈

- **基础理论**：机器学习、深度学习、大语言模型
- **开发框架**：LangChain、OpenAI API、Anthropic Claude
- **开发工具**：GitHub Copilot、Cursor、AI 辅助工具
- **应用场景**：智能对话、代码生成、内容创作
