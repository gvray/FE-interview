# 移动端适配

移动端 H5 适配的核心知识：1px 问题、dpr、rem/vw、300ms 延迟、键盘遮挡、触摸反馈等。配套 [移动端适配 FAQ](./FAQ.md) 汇总了高频问题与解法。

跨端框架见 [跨端框架](../taro/)，小程序见 [小程序](../miniprogram/)。

## 学习路径

1. 理解 dpr、物理像素 vs CSS 像素、viewport。
2. 掌握 rem / vw 适配方案与 1px 问题解法。
3. 处理 300ms 延迟、点击穿透、键盘遮挡输入框等常见坑。
4. 关注移动端性能（滚动卡顿、图片模糊、长按闪退）。
